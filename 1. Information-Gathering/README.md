# Information Gathering

Information Gathering (Reconnaissance) is the first phase of any penetration test. It involves collecting as much information as possible about a target.

> More information = Higher chances of successful exploitation in later stages.

---

## 🔍 Types of Information Gathering

### 1. Passive Information Gathering
- No direct interaction with the target
- Target unaware of reconnaissance activity

### 2. Active Information Gathering
- Direct interaction with the target
- Can be detected/logged by target systems

---

## 🎯 Objectives

- Identify target infrastructure
- Discover exposed services
- Gather publicly available information (OSINT)
